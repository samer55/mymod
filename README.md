# 项目运行

```
git clone https://github.com/shooterRao/react-native-fruitStore.git 或者直接下载

cd react-native-fruitStore

yarn

android: react-native run-android
ios: react-native run-ios
```

# 实现功能

* 商品预览
* 加入购物车
* 商品的增删查改
* 模拟支付
* 清空购物车，还原商品初始状态
* 模拟下拉加载
* 页面的跳转和传值
* 查看订单列表(2018/1/8 增加)
